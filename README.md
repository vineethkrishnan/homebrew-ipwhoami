# homebrew-ipwho

Homebrew tap for [ipwho](https://github.com/vineethkrishnan/ipwho) — IP geolocation lookup from your terminal.

## Install

```bash
brew tap vineethkrishnan/ipwho
brew install ipwho
```

## Update

```bash
brew upgrade ipwho
```

## Uninstall

```bash
brew uninstall ipwho
brew untap vineethkrishnan/ipwho
```
